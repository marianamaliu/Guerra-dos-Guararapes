# Guerra-dos-Guararapes
Guerra dos Guararapes ambienta batalha de Pernambuco contra holandeses no século XVII. Comando do exército local utiliza ícones como Zumbi e Cangaceiro contra artilharia invasora. Mecânica central exige aplicar Merge Sort ou Quick Sort para organizar combatentes. Ordenação estratégica de valores numéricos define formação ideal no front.
